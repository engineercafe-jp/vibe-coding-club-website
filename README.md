# バイブコーディングクラブ ウェブサイト

GitHub Pages にデプロイするバイブコーディングクラブのランディングページ

## 概要

バイブコーディングクラブのイベントで実装されたプロダクトを一覧表示するランディングページ。Apple の App Store のようなプロダクトギャラリーを中心とした、モバイルファーストのシンプルなウェブサイト。

## 機能

### ヘッダー

- 左: サイトタイトル「バイブコーディングクラブ」
- 右: 申し込みリンク

### ファーストビュー

- イベント情報（`event.md` から取得）
- Google Map のフレームによるアクセス情報
- オンライン見学リンク（connpass / Peatix / TECH PLAY）

### プロダクト一覧

App Store 風のカードレイアウトでプロダクトを表示する。

#### プロダクトカード

| 項目 | 内容 |
|------|------|
| SVG アイコン | プロダクト概要に基づくフラットデザイン |
| タイトル | プロダクト名 |
| 概要 | 20 文字以内 |
| 使用技術 | 技術スタック |
| 実装者 | GitHub ユーザー ID |
| 登録日時 | プロダクトの登録日時 |

#### プロダクトモーダル

カードクリックで開くモーダルに、カード情報に加えて以下を表示する。

| 項目 | 内容 |
|------|------|
| 詳細 | 500 文字以内 |
| GitHub URL | リポジトリへのリンク |

### SVG アイコン

- プロダクト概要に基づいてシンプルなフラットデザインで作成
- アイコンの色は実装者の GitHub ユーザー ID から一意に決定

## データソース

| ファイル | 内容 |
|----------|------|
| `event.md` | ファーストビューに表示するイベント情報 |
| `product.md` | プロダクト一覧の情報 |

プロダクトの実装はサブモジュールとしてプルし、解析する。

## 技術要件

- モバイルファースト
- GitHub Pages でホスティング可能なシンプルな構成
- プロフェッショナルなデザイン

## デプロイ

GitHub Pages にデプロイする。
